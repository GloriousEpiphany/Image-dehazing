# 非均质有雾图像去雾系统

## 项目简介

本项目是一个基于Python和PyQt6开发的非均质有雾图像去雾应用程序，采用自研的改进暗通道先验算法，能够有效处理不同浓度和分布的雾霾图像，恢复清晰的场景细节。

## 主要特性

- 🖼️ **直观的图形界面**：基于PyQt6开发的现代化用户界面
- 🔬 **自研去雾算法**：改进的非均质暗通道先验算法
- 📊 **实时进度显示**：处理过程可视化和进度跟踪
- ⚙️ **参数可调节**：支持算法参数的自定义调整
- 📁 **多格式支持**：支持JPG、PNG、BMP、TIFF等常见图像格式
- 💾 **处理历史**：自动保存处理记录和参数配置
- 🛠️ **错误处理**：完善的异常处理和用户提示机制

## 系统要求

- Python 3.8+
- Windows 10/11 (推荐)
- 内存：至少4GB RAM
- 存储：至少500MB可用空间

## 算法原理

本系统采用改进的非均质暗通道先验算法，主要包括以下步骤：

1. **暗通道计算**：计算输入图像的暗通道图
2. **大气光估计**：基于暗通道和亮度信息估计大气光值
3. **透射率估计**：计算初始透射率图
4. **透射率优化**：使用引导滤波优化透射率图
5. **非均质调整**：根据雾霾密度分布进行自适应调整
6. **场景辐射恢复**：恢复无雾的场景辐射
7. **后处理增强**：对结果进行色彩和对比度增强

## 配置说明

### 算法参数

- **omega (ω)**：暗通道权重系数 (0.1-1.0，默认0.95)
- **t0**：透射率下限 (0.01-0.5，默认0.1)
- **A**：大气光值 (自动估计或手动设置)
- **r**：暗通道窗口半径 (1-15，默认15)
- **eps**：引导滤波正则化参数 (0.001-0.1，默认0.0001)

## 故障排除

### 常见问题

**Q: 程序启动失败**
A: 检查Python版本和依赖包是否正确安装

**Q: 处理速度慢**
A: 尝试降低图像分辨率或调整算法参数

**Q: 结果不理想**
A: 根据图像特点调整omega和t0参数

**Q: 内存不足**
A: 关闭其他应用程序或处理较小的图像
